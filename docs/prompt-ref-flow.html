Key Insight
prompt_ref is a single string — e.g. "agent-researcher" — that threads through all three systems. It's the name when creating prompts in the Registry, the prompt_id stored in LiteLLM, and the lookup key that cs-agent-service uses at startup to resolve each role's system prompt. One string, three repos, zero coupling beyond a naming convention.
